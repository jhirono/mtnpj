Awesome Climbing Search

Welcome to Awesome Climbing Search! This application helps climbers find routes that match their preferences across multiple climbing areas.

About

Awesome Climbing Route Search includes:

  • Filtering by grade, type, and various tags
  • Sorting options (grade, stars, votes, left-to-right)
  • Area-based filtering with hierarchical selection

Current Areas

The application includes routes from:

  • Indian Creek
  • Squamish
  • Joshua Tree National Park
  • Leavenworth
  • Index
  • Red Rocks
  • Yosemite National Park

How to Use

Getting started with Awesome Climbing Search is easy:

  1. Select Areas: Use the area dropdown to choose specific climbing areas you're interested in. You can select multiple areas or entire regions.
  2. Filter Routes: Narrow down your search using the filters panel:
    • Set grade range (e.g., 5.8 to 5.10c)
    • Choose route types (Sport, Trad)
    • Select specific tags for style, holds, conditions, etc.
  3. Sort Results: Arrange routes by:
    • Grade (easiest to hardest or vice versa)
    • Stars (highest rated first)
    • Popularity (most climbed first)
    • Left-to-right position at the crag
  4. Browse Routes: Scroll through the results to see route details including grade, stars, and tags. The app will automatically load more routes as you scroll down.
  5. View Details: Click on route names to see full descriptions, protection information, and comments from other climbers.

The application is designed to help you quickly find routes that match your preferences, whether you're planning a trip to a new area or looking for specific types of climbs.
